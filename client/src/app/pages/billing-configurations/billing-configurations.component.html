Included Radius
This is the local distance included in normal pricing. If a request is inside this radius, no extra travel charge is added.
Rate / km
Once the included radius is crossed, this is the additional amount charged for every extra kilometre.
Max Auto-Match
This is the hard distance limit for automatic dispatch. If the request is beyond this distance, the platform should not auto-notify the guard.
Manual Review Over
Use this as an escalation threshold. Long-distance requests above this level should be checked by operations before they move forward.
